Bootstrap 5 Grids system

Bootstrap's grid system is built with flexbox and allows up to 12 columns across the page

If you do not want to use all 12 columns individually, you can group the columns together to create wider columns.

.col (extra small devices - screen width less than 576px)

.col-sm (small devices - screen width to or greater than 576px)

.col-md (Medium devices) - screen width equal to or greater than 768px

.col-lg (Large devices) - screen width equal to or greater than 992px

.col-xl (extra large devices) - screen width equal to or greater than 1200px

.col-xxl (xxlarge devices) - screen width equal to or greater than 1400px

The classes can be combined to create more dynamic and flexible layouts

Tip: Each class scales up, so of you want to set the same widths for sm and md, you only need to specify sm.
