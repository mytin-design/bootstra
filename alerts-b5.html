BOOTSTRAP 5 ALERTS

Alerts

Bootstrap 5 provides an easy way to create predefined alert messages:

Success! This alert box could indicate a successful or positive action.

Info! This alert box could indicate a neutral informative change or action.

Warning! This alert box could indicate a warning that might need attention.

Alert Links

Add the .alert-link class to any links inside the alert box to create "matching colored links";

Closing Alerts

To close the alert message, add a .alert-dismissable class to the alert container. Then add class="btn-close" and data-bs-dismiss="alert" to a link or a button element (when you click on this the alert box will disappear).
