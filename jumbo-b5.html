Bootstrap 5 Jumbotron

A jumbotron was introduced in Bootstrap 3 as a big padded box for calling extra attention to some special content or information.

Jumbotron are no longer supported in Bootstrap 5. However, you can use a div element and add special helper classes together with a color class to achieve the same effect:
